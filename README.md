# CSCI 164 Project

This project compares supervised learning models across three datasets using scikit-learn.

## Datasets
- Wisconsin Diagnostic Breast Cancer
- Dry Bean
- Diamonds

## Models Used
- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- MLPClassifier
- Linear Regression
- KNN Regressor
- Decision Tree Regressor

## Files
- `notebook/final_ml_project_notebook.ipynb`
- `report/CSCI 164 Project Report.pdf`
- `presentation/CSCI 164 Project.pptx`

## Summary
The project explores preprocessing, baseline modeling, hyperparameter tuning, evaluation, and comparison to prior work across classification and regression tasks.
